# TAS

#### Preparation

1. Download Dataset (RefCOCO, RefCOCO+, RefCOCOg) and put in your path

2. Change the configuation in "./config/refcoco*/refcoco*.yaml". Mainly the "DATA.data_dir", "TRAIN.clip_pretrain", "TRAIN.pretrained_lm" and "TRAIN.output_folder"

3. Prepare SAM-H, CLIP and bert-base-uncased model

4. Prepare captions for images (Using MLLM) and put them in ./caption_llm/

5. Install the environment requirements (pip install -r requirements.txt). For syntactic parsing tools, you need to manually install some extension (en-core-web-trf in spacy, wordnet in nltk) 

#### Usage
```
python llm_tas_main.py --config config/refcoco/refcoco_val.json
```

You could use other captions generated by other MLLMs. You just change the code refer to llm_tas_main_\{model_name\}.py
